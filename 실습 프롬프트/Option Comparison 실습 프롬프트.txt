[Option Comparison 실습 프롬프트]

아래의 3가지 대안을 Option Comparison Framework에 따라 비교해줘.

옵션 리스트:
A) AI 기반 문서 자동 생성 시스템 도입
B) 사내 검색 시스템 업그레이드
C) 기존 프로세스 유지 + 매뉴얼 개선

다음 단계에 따라 분석해줘.

1단계) 의사결정 기준 설정

아래 기준을 사용해 각 기준의 중요도도 1~5점으로 평가해줘.

비용(Cost)
효과(Impact)
실행 용이성(Ease of Implementation)
위험도(Risk)
장기 전략 적합도(Strategic Fit)

출력: 기준 + 중요도 점수 + 선택 이유

2단계) 옵션별 평가

각 옵션(A,B,C)을 위 기준(총 5개)으로 각각 1~5점으로 평가해줘.
(평가 근거 반드시 포함)

형식 예시:

비용: 2점 (초기 구축비용 높음)
효과: 5점 (문서 작성 시간 크게 단축)
…

3단계) 가중치 합산

1단계에서 설정한 중요도를 곱해서
각 옵션의 가중 합산 점수(Total Weighted Score)를 계산해줘.

형식 예시:
옵션 A 총점 = Σ(기준 점수 × 중요도)

4단계) 우선순위 Recommendation

각 옵션의 총점을 기반으로

최적 선택(1순위)
조건부 선택(2순위)
비권장 선택(3순위) 으로 정리해줘.

5단계) 실행 전략

최종 선택 옵션에 대해
바로 실행해야 할 3가지 Action Plan
리스크 대비 방안을 제시해줘.

출력 형식:

의사결정 기준표
옵션 평가표
가중치 합산표
최종 우선순위
실행 전략 요약
